# Global Fashion Retail Analytics & Machine Learning Project

An end-to-end retail analytics and business intelligence project built using *Power BI, SQL, Python, and Machine Learning* to analyze global fashion retail operations, customer behavior, sales performance, and return trends.

This project simulates a real-world enterprise analytics workflow followed by modern data analysts and BI professionals — starting from raw transactional data to interactive dashboards, SQL analysis, predictive analytics, and business insights generation.

---

# Project Objectives

The primary objectives of this project were:

- Analyze global retail sales performance
- Track revenue, returns, and customer metrics
- Identify high-performing product categories
- Understand geographic sales distribution
- Evaluate customer purchasing behavior
- Perform return rate analysis
- Build forecasting and machine learning workflows
- Create professional business dashboards for decision-making
